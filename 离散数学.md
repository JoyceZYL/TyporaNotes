## 集合论



#### 集合 Set

定义：**<font color=likblue>指定范围</font>** 内 **<font color=likblue>满足给定条件</font>** 的**<font color=likblue>所有对象</font>**

##### 集合的表示

```
A = {1，2，x，n^2，...}

B = {x| x < 5}
```

##### 集合基数 Base Number

- 集合A包含的元素数目
- 表示为|A|
- A包含n个元素，A为n元集。包含m个元素的A的子集，称为m元子集。
- n元集共有2^n^的子集，成为幂集

## 数学归纳法

**Mathematical Induction**

1. 证明 P(1) 是正确的
2. 证明在 P(n) 正确的情况下，P (n+1) 是正确的
3. 所以对于所有 n 来说，P(n) 都是正确的
